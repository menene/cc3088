Restricciones en SQL

NOT NULL

Ejemplo de valores permitidos:

  • "Juan Pérez" ✅
  • "" ❌ (No permitido, no puede ser nulo)

UNIQUE

Ejemplo de valores permitidos:

  • "correo1@example.com" ✅
  • "correo2@example.com" ✅
  • "correo1@example.com" ❌ (No permitido, ya existe)

CHECK

Ejemplo de valores permitidos:

  • Edad: 25 ✅
  • Edad: 17 ❌ (No permitido, debe ser mayor o igual a 18)

DEFAULT

Ejemplo de valores por defecto:

  • Estado del pedido: "pendiente" ✅ (valor por defecto si no se especifica otro)
  • Estado del pedido: "en proceso" ✅

Introducción a SQL

Historia y Características

  • SQL fue desarrollado en la década de 1970 en IBM.
  • Es un lenguaje declarativo utilizado para la gestión de bases de datos relacionales.
  • Permite definir, manipular y consultar datos.

Características de SQL

  • Lenguaje estándar para bases de datos relacionales.
  • Independiente del sistema operativo y del motor de base de datos.
  • Soporta integridad referencial y transacciones.

Primeras Consultas en SQL

SELECT todo

SELECT * FROM empleados;

SELECT con columnas específicas

SELECT nombre, edad FROM empleados;

SELECT con WHERE

SELECT * FROM empleados WHERE edad > 30;
